| プレビュー | CTkScrollableFrame + CTkLabel | Markdown→HTMLのリアルタイムプレビュー |
| SEOスコアパネル | CTkFrame | SEOスコアとチェック結果を表示 |
| SEOスコア表示 | CTkLabel + CTkProgressBar | 100点満点のスコアと色分け評価 |
| SEOチェックリスト | CTkScrollableFrame | 各SEO項目の達成状況を表示 |
| SEO改善提案ボタン | CTkButton | LLMによるSEO改善提案を生成 |
| SEO対策ポイントパネル | CTkScrollableFrame | プロトタイプ生成時のSEO対策解説を表示 |
| SEO対策ポイント項目 | CTkFrame | カテゴリ別の施策・理由・編集時注意を表示 |
| すべて表示ボタン | CTkButton | SEO対策ポイントの全項目表示/折りたたみ切替 |
| 再生成ボタン | CTkButton | LLMで記事を再生成 |
| 元に戻すボタン | CTkButton | プロトタイプの状態に復元 |
| 下書き保存ボタン | CTkButton | 現在の状態を保存 |
| 投稿ボタン | CTkButton | SCR-006へ遷移 |

#### SEOパネル動作仕様

- SEOスコアは記事編集中にリアルタイムで更新（1秒デバウンス）
- 各チェック項目はアイコンで状態を表示（✅ 達成 / ⚠ 改善推奨 / ❌ 未達成）
- チェック項目クリックで詳細と改善方法をツールチップ表示
- 「SEO改善提案」ボタンでLLMによる具体的な改善案をダイアログ表示
- SEOパネルは表示/非表示を切り替え可能（Ctrl+Shift+E）

#### SEO対策ポイントパネル動作仕様

- プロトタイプ生成時にLLMが出力したSEO対策ポイント解説を表示する
- 各ポイントはカテゴリ別（タイトル、見出し構造、キーワード配置等）に表示
- 各ポイントには3つの情報を表示: ✏ 施策内容 / 💡 理由・効果 / ⚠ 編集時の注意
- 初期表示は先頭2〜3カテゴリのみ、「すべて表示」ボタンで全カテゴリを展開
- パネルはSEOスコアパネルの下部に配置（折りたたみ可能）
- 記事を再生成した場合、SEO対策ポイント解説も新しい内容に更新される
- SEO対策ポイントはユーザーが記事を編集する際の参照情報として常時閲覧可能

#### 動作仕様

- エディタとプレビューは左右分割（ドラッグでリサイズ可能）
- SEOパネルはプレビュー下部に表示（折りたたみ可能）
- エディタの変更は300msのデバウンス後にプレビューに反映
- 30秒ごとに自動保存
- Ctrl+S で手動保存
